# lazydf - Serialization to binary (deterministic and safe  - when possible) 
 


## Python installation
### from package
```bash
# Set up a virtualenv. 
python3 -m venv venv
source venv/bin/activate

# Install from PyPI
pip install lazydf
```

### from source
```bash
git clone https://github.com/davips/lazydf
cd lazydf
poetry install
```
